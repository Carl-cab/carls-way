### Scenario 1: Idempotency via balance_processed_at

**Setup Commands:**
```bash
# Record baseline
psql $DATABASE_URL << 'SQL'
  SELECT email, balance_usd FROM users WHERE email='test-us-001@validation.local';
SQL
# Expected: test-us-001@validation.local | 5000.00

# Send first settled webhook
curl -X POST http://localhost:3000/api/webhooks/plaid \
  -H "Content-Type: application/json" \
  -H "Plaid-Verification: <valid-jwt>" \
  -d '{"webhook_type":"TRANSFER","webhook_code":"STATUS_UPDATE","data":{"transfer_id":"transfer_id_001","status":"settled"}}' \
  | jq '.balance_result | {success, balanceUpdated, amountApplied}'

# Check after first webhook
psql $DATABASE_URL << 'SQL'

# Expected: test-us-001@validation.local | 5100.00

```

- ✅ balance_usd: 5000 → 5100 (first call)
