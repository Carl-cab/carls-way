# BCFNJC IT Onboarding Video — Executive Overview

**Organization:** BC First Nations Justice Council (BCFNJC)  
**Project Type:** Microsoft 365 IT Onboarding Video  
**Delivery Format:** SharePoint-hosted MP4 with closed captions  
**Estimated Runtime:** 12–15 minutes  
**Primary Audience:** New BCFNJC employees  

---

## Purpose

Deliver a polished, privacy-safe onboarding experience that walks new staff through Microsoft 365 tools, hardware care, IT support processes, and organizational digital resources — without exposing live sensitive data.

## Core Constraints

| Constraint | Approach |
|---|---|
| No live screen recordings | Animated screenshots with masked sensitive data |
| No exposed employee data | Privacy review gate before every asset enters production |
| Remote-accessible | SharePoint + Microsoft Stream hosting |
| Accessible | Closed captions (VTT), WCAG 2.1 AA contrast |
| Indigenous respectful | Calm, community-centered tone; no stereotypes |

## Key Deliverables

1. Fully scripted and narrated video (MP4, 1920×1080, H.264)
2. Closed captions file (WebVTT)
3. SharePoint embed page
4. Production asset folder (organized per naming convention)
5. QA-approved final cut

## Stakeholders

| Role | Responsibility |
|---|---|
| IT Representative (BCFNJC) | Screenshot capture, privacy review, technical accuracy |
| Project Lead / Producer | Coordination, timeline, approvals |
| Instructional Designer | Script, storyboard, narration |
| Video Editor | Animation, assembly, export |
| Accessibility Lead | Caption review, contrast check |
| Manager Approval | Final sign-off before publish |
