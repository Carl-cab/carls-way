# BCFNJC Onboarding Video — Screenshot Capture Guide
**For: IT Representative**  
**Purpose:** Step-by-step instructions for capturing clean, production-ready screenshots

---

## Before You Start

### Setup Checklist
- [ ] Use a **test/demo account** — never capture from your personal or admin account
- [ ] Set display resolution to **1920×1080** (or highest available)
- [ ] Set Windows scaling to **100%** (Settings → Display → Scale = 100%)
- [ ] Use **Microsoft Edge** in a clean profile (no personal bookmarks visible unless required)
- [ ] Close all unrelated apps — no notification badges, no personal apps in taskbar
- [ ] Sign into demo account in all M365 apps before capturing
- [ ] Clear notification badges (Teams, Outlook) so they don't appear in shots
- [ ] Set system clock to current date/time (fine to show)
- [ ] Turn on **Do Not Disturb** / Focus Assist to prevent pop-up notifications during capture

### Demo Account Setup
Create or use an existing IT demo/test account:
- Display name: "New Employee" or "[First Name] [Last Name]" (a placeholder, not a real person)
- Email: something generic like `newstaff@bcfnjc.com` or `demo@bcfnjc.com`
- This avoids exposing any real employee's account in screenshots

## Capture Tool Options

### Option A — Snipping Tool (Windows, free)
- Press `Windows + Shift + S`
- Select "Window Snip" for clean app captures
- Auto-saves to clipboard; paste into Paint/Photoshop to save

### Option B — Snagit (Recommended, ~$63 one-time)
- Best for scrolling captures, annotations, and quick blur/redaction
- Capture → save directly to PNG
- Built-in blur tool for sensitive data

### Option C — ShareX (Free, open source)
- Highly configurable, region/window/full-screen modes
- Can auto-upload and annotate

**Recommended format:** PNG, 1920×1080 or native app window width

## After Capture — Privacy Masking Steps

1. Open each RAW screenshot in **Snagit**, **Photoshop**, or **Canva**
2. Apply the Privacy Checklist from `08_SECURITY_COMPLIANCE.md`
3. Blur all sensitive areas (Gaussian blur, 20–30px)
4. Save masked version with `_MASKED` suffix
5. Send to producer for second review
6. Once approved, rename to `_APPROVED` and move to `/01_Screenshots/APPROVED/`
