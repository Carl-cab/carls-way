# BCFNJC Onboarding Video — Visual Style Guide

---

## Design Philosophy

The video should feel like BCFNJC itself: professional, trustworthy, and human. It reflects an organization that takes its work seriously while remaining warm, accessible, and connected to the communities it serves.

> "Modern enough to feel current. Calm enough to feel welcoming. Grounded enough to feel real."

## Typography

### Primary Font: Montserrat
*(This aligns with BCFNJC's established brand standard — also used in Word templates)*

| Style | Weight | Size | Usage |
|---|---|---|---|
| Display / Title | ExtraBold (800) | 48–64pt | Main title card |
| Section Heading | Bold (700) | 32–36pt | Section title cards |
| Lower Third | SemiBold (600) | 22–26pt | Scene label bar |
| Body / Callout | Medium (500) | 16–18pt | Callout labels, bullet text |
| Caption | Regular (400) | 14pt | Closed caption text |

### Fallback Font: Inter / Segoe UI
- Use if Montserrat is unavailable in the production tool
- Both are clean, readable, modern sans-serifs

---

## Layout Grid

### Safe Area
- Keep all text and UI elements within a 10% margin from all edges (192px left/right, 108px top/bottom at 1920×1080)
- Logo: always top-left corner, within safe area

### Screenshot Placement
- Full-width screenshots: scale to fill frame, allow 5% overflow for Ken Burns if needed
- PiP screenshot (with avatar): screenshot fills left 65%, avatar in lower-right 30%
- Never crop screenshots in a way that removes essential UI chrome

---

## Iconography Style

- Use outline-style icons (not filled) for a clean, modern look
- Consistent stroke weight: 2px
- Recommended icon set: Fluent UI Icons (Microsoft's own icon system — perfect for M365 content)
- Alternative: Material Icons (Google) — clean, widely available
- Icons should be white on colored backgrounds, or brand-colored on white backgrounds

---

## Motion & Animation Style

| Element | Style |
|---|---|
| Transitions | Smooth, never abrupt — ease-in-out on all |
| Timing | Deliberate — not rushed (training videos need time to process) |
| Energy level | Calm, measured — not flashy or aggressive |
| Zoom speed | Slow and purposeful (1–2s for any significant zoom) |
| Text reveals | Fade in or slide up — never spin, bounce, or pop |
| Callout draws | Stroke-on animation — feels handcrafted and direct |

## Indigenous Respectful Design Principles

BCFNJC serves First Nations communities in BC. The video's visual language should:

- **Avoid stereotypes:** Do not use generic "Indigenous" design motifs (feathers, totem-inspired patterns) as decorative filler. If cultural design elements are used, they must be approved by BCFNJC's leadership and sourced appropriately.
- **Use the organization's own branding:** Let BCFNJC's official brand guide lead all design decisions.
- **Prioritize dignity and professionalism:** The same visual respect extended to any professional organization.
- **Be community-centered in tone:** Inclusive, welcoming, clear — never condescending or overly corporate.
- **Representative imagery:** If stock photos are used (e.g., for hardware scenes), select images that reflect diversity and are not stereotyping.
- **Language:** Plain language, no jargon. Accessible to people with varying tech backgrounds.

## Background Music Direction

- **Genre:** Ambient / Lo-fi / Corporate calm
- **Tempo:** 60–80 BPM (relaxed, not urgent)
- **Instrumentation:** Light piano, soft pads, gentle acoustic guitar — no drums or heavy beats
- **Volume:** -28 LUFS (well under narration at -14 LUFS) — music is felt, not heard
- **Suggested sources:** Pixabay (free), Artlist (licensed), Epidemic Sound (licensed)
- **Avoid:** Music with lyrics, anything that feels energetic or corporate-sales-y

---

## Logo Usage

- Always use the full BCFNJC logo with clearspace (minimum 50px clearspace on all sides at 1080p)
- Do not stretch, rotate, or recolor the logo
- On dark backgrounds: use the white/light version
- On light backgrounds: use the full-color or dark version
- Logo appears: title card (Scene 01), end card (Scene 12), and optionally as a small watermark in lower-left of PiP layouts

---

## Do / Don't Summary

| Do | Don't |
|---|---|
| Use clean, minimalist layouts | Overcrowd slides with text |
| Animate purposefully | Add animation for decoration |
| Use Montserrat consistently | Mix multiple font families |
| Keep color palette tight | Introduce new colors per scene |
| Use blur for transitions | Use hard cuts between screenshots |
| Maintain safe area margins | Let important content touch edges |
| Use subtle amber for callouts | Use red (implies error) for highlights |
| Keep music under narration | Let music compete with voice |
