# BCFNJC Onboarding Video — Production Plan

---

## Production Phases

### Phase 1 — Pre-Production (Days 1–5)
- [ ] Kickoff meeting with IT representative and project lead
- [ ] Confirm script accuracy with IT team
- [ ] Define visual style (colors, fonts, lower-third style)
- [ ] Identify all screenshots required (see storyboard)
- [ ] IT rep captures raw screenshots
- [ ] Privacy review: blur/mask all sensitive data
- [ ] Organize assets using folder structure (see Templates)
- [ ] Record or generate voiceover narration
- [ ] Select and configure AI avatar (if using Synthesia/HeyGen)

### Phase 2 — Production (Days 6–12)
- [ ] Import approved screenshots into animation tool (Camtasia/Canva)
- [ ] Apply animations: zoom, cursor paths, click indicators, callouts
- [ ] Assemble scenes in sequence per storyboard
- [ ] Insert avatar segments (intro, transitions, outro)
- [ ] Sync narration audio to visuals
- [ ] Add lower thirds, section titles, on-screen text
- [ ] Apply BCFNJC branding (colors, logo placement)
- [ ] Add background music (subtle, royalty-free)

### Phase 3 — Post-Production (Days 13–16)
- [ ] Internal review cut (rough cut)
- [ ] IT accuracy review by BCFNJC IT rep
- [ ] Revisions round 1
- [ ] Accessibility review (captions, contrast)
- [ ] Final cut approval by manager
- [ ] Export: MP4 1920×1080, H.264, AAC audio
- [ ] Generate VTT caption file
- [ ] Upload to SharePoint / Microsoft Stream

### Phase 4 — Deployment (Days 17–18)
- [ ] Upload to Stream or SharePoint document library
- [ ] Embed on SharePoint onboarding page
- [ ] Set permissions: All Staff (view-only)
- [ ] Test playback on desktop and mobile
- [ ] Announce to HR/IT for inclusion in onboarding flow

## Asset Requirements

### Screenshots Needed (full list in Storyboard)
- BCFNJC logo / branding assets
- Laptop photo (generic, no serial numbers visible)
- Microsoft Edge — bookmarks bar, Intranet homepage
- OneDrive — File Explorer view, folder structure
- Outlook — compose window, address book, calendar view (all data masked)
- Microsoft Teams — main interface, notification settings, auto-launch setting
- Microsoft Word — File > New > Templates view, Montserrat font selection
- Adobe Acrobat Pro — PDF open, sign tool visible
- SharePoint — BCFNJC shared document library
- IT Support contact info graphic (custom-designed, not a screenshot)

### Audio Assets
- Narration audio (WAV or MP3, 44.1 kHz, stereo)
- Background music track (royalty-free, calm/ambient)
- Optional: subtle UI click sound effects

### Graphic Assets
- BCFNJC logo (PNG, transparent background)
- Color palette (see Visual Style Guide)
- Section title card templates
- Lower-third name/role template
- Privacy blur/redaction overlay

## Privacy Validation Workflow

```
IT Rep captures screenshot
        ↓
Privacy Checklist applied (see Security Checklist)
        ↓
Blur/redact: names, emails, file names, calendar entries, chat messages
        ↓
Producer spot-check review
        ↓
Asset approved → moves to production folder
        ↓
Final QA before export: second pass on all frames
```
