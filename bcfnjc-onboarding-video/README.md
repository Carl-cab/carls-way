# BCFNJC IT Onboarding Video — Production Blueprint

**Organization:** BC First Nations Justice Council (BCFNJC)  
**Project:** Microsoft 365 Employee IT Onboarding Video  
**Status:** Pre-Production — Blueprint Complete  

---

## What This Is

A complete production blueprint for creating a professional Microsoft 365 IT onboarding video for new BCFNJC employees. The video uses animated screenshots (not live recordings) to protect sensitive organizational data, with an AI avatar presenter, professional narration, and SharePoint hosting.

**Estimated video runtime:** 12–15 minutes  
**Format:** MP4 1920×1080, hosted on Microsoft Stream / SharePoint

## Quick Start by Role

### If you're the IT Representative
1. Read `08_SECURITY_COMPLIANCE.md` — understand what must be masked
2. Read `11_SCREENSHOT_CAPTURE_GUIDE.md` — set up demo account, capture screenshots
3. Review `02_STORYBOARD.md` — confirms which screenshots are needed per scene
4. Deliver approved (masked) screenshots to producer

### If you're the Video Editor
1. Read `02_STORYBOARD.md` — understand each scene's visual requirements
2. Read `04_ANIMATION_GUIDE.md` — animation techniques
3. Follow `12_CAMTASIA_BUILD_GUIDE.md` — step-by-step build instructions
4. Use `09_TEMPLATES.md` for folder structure and file naming
5. Import `13_CAPTIONS_SAMPLE.vtt` as your caption starting point

### If you're the Producer / Project Lead
1. Start with `00_EXECUTIVE_OVERVIEW.md` and `01_PRODUCTION_PLAN.md`
2. Use `06_SOFTWARE_STACK.md` to make tool decisions
3. Use `09_TEMPLATES.md` approval checklist to track progress
4. Use `07_SHAREPOINT_DEPLOYMENT.md` for the publish step

### If you're setting up the AI Avatar / Narration
1. Read `05_AVATAR_STRATEGY.md` — platform recommendations
2. Use `03_NARRATION_SCRIPT.md` — paste directly into Synthesia/ElevenLabs
3. Check `10_VISUAL_STYLE_GUIDE.md` — avatar framing and tone guidance

## Recommended Tool Stack (Mid-Range Budget)

| Task | Tool |
|---|---|
| AI Avatar | HeyGen or Synthesia |
| Narration | ElevenLabs or Azure Neural TTS |
| Screenshot animation | Camtasia |
| Final editing | Camtasia or Clipchamp |
| Captions | Descript → export VTT |
| Hosting | Microsoft Stream + SharePoint |
| Privacy masking | Snagit or Photoshop |

## 18-Day Production Timeline

```
Days 1–5:   Pre-Production (script review, screenshots, audio)
Days 6–12:  Production (animation, assembly, avatar compositing)
Days 13–16: Post-Production (review, revisions, captions, QA)
Days 17–18: Deployment (Stream upload, SharePoint embed, announce)
```
